Welcome to the wavgen.ca project!

This is a personal website created by Rob McDonald, hosted on GitHub Pages. It's a playground for exploring web development, programming, and integrations with modern tools. The site features content on Quantum and Cloud Computing, Real-time Generative 3D AV, and Visual Art.

Project Structure:

  • CSS and Styling: The css directory contains the main stylesheet, wavgen.css, which defines the look and feel of the site.
  • Images: The images directory holds various assets, including a parallax background and some unique toe images.
  • JavaScript: The js directory includes scripts for loading components and animations, with GSAP plugins enhancing interactivity.
  • Components: The components directory has subnav files for different sections like art, data, music, and videos, helping organize the site's navigation.
  • Main HTML Files: index.html is your starting point, while indexbackup.html serves as a backup. There's also a 404.html for error handling.

Getting Started:

Since you've already cloned the project, dive into the index.html to see how the site is structured. Explore the components directory to understand how different sections are organized. If you're interested in animations, check out the GSAP scripts in the js directory.
