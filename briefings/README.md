# Daily briefings

One markdown file per day, generated by the daily news loop. Named `YYYY-MM-DD.md`.

The loop also reads the last 7 days of briefings to dedup news items, so don't bulk-rename or delete files unless you're OK with possible re-coverage.

## Schema

```markdown
---
date: 2026-05-22
sections:
  music: 4
  video: 3
  data: 5
  art: 2
---
```
