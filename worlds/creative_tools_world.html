Creative Workflows

Pipeline Explorer

Four end-to-end creative production pipelines showing how tools connect in sequence — from concept to output.

Pipeline 01
Generative Visual Pipeline
p5.js sketch → WebGL render → TD import → output

Sketch generative forms in p5.js, route geometry to a WebGL fragment shader for GPU post-processing, then pull the rendered texture into TouchDesigner for projection or video output.

Pipeline 02
Audio-Reactive Pipeline
Max/MSP audio → OSC → TD / p5.js → visual

Max/MSP analyses or generates audio in real-time. Analysis data (amplitude, spectrum, beat) streams via OSC to TouchDesigner or p5.js, driving generative visuals in sync with sound.

Pipeline 03
AI-Assisted Pipeline
Python / ML → image output → TD texture → WebGL post

Python ML models (StyleGAN, diffusion, NeRF) generate image or video frames. TouchDesigner ingests as live texture. WebGL post-processing (GLSL shaders) applies real-time stylization.

Pipeline 04
Live Performance Pipeline
Max/MSP → OSC → TD network → projection

Performer triggers Max/MSP patches live. OSC messages stream over network to TouchDesigner across multiple machines. Output feeds multi-channel projection mapping in real-time.
