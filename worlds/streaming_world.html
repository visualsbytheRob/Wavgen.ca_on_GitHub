Section 09

Streaming Glossary

22 essential terms

Bitrate
Data transferred per second in a stream, measured in kbps or Mbps. Higher bitrate = better quality but more bandwidth.
Buffer
Temporary storage of stream data allowing smooth playback despite network fluctuations.
CBR
Constant Bitrate — encoder maintains a fixed output bitrate regardless of scene complexity.
CDN
Content Delivery Network — distributed server infrastructure that delivers streams to viewers worldwide with low latency.
Chroma Key
Green/blue screen technique that removes a solid color background to composite subjects onto virtual backdrops.
Codec
Encoder/decoder algorithm (e.g., H.264, AV1) that compresses video for transmission and decompresses for playback.
Delay
Intentional stream delay added by broadcasters for content moderation or to prevent stream sniping in competitive games.
Encoder
Hardware or software that converts raw video/audio into a compressed format suitable for streaming (OBS, x264, NVENC).
Frame Rate
Number of frames per second (fps). 30fps is standard; 60fps provides smoother motion especially in fast-paced games.
H.264
The most widely supported video codec. Excellent compatibility and quality/bitrate balance for streaming at 1080p60.
Ingest
The platform server that receives the streamer's upload. Ingest URL is configured in OBS or streaming software.
Keyframe
A fully encoded frame (I-frame) used as reference points. Streaming platforms require keyframe intervals of 2 seconds.
Latency
Delay between what the streamer does and what viewers see. Ranges from sub-second (WebRTC) to 30+ seconds (DVR mode).
NVENC
NVIDIA's hardware video encoder. Offloads encoding from CPU to GPU, allowing high-quality streams with minimal CPU impact.
OBS
Open Broadcaster Software — the free, open-source standard streaming application used by the majority of streamers worldwide.
Packet Loss
Network packets that fail to reach their destination, causing dropped frames, visual artifacts, or stream disconnections.
RTMP
Real-Time Messaging Protocol — the universal standard for sending streams from encoder to platform ingest servers.
Scene
In OBS, a collection of sources (camera, screen, overlays) configured as a layout that can be switched during broadcast.
SRT
Secure Reliable Transport — modern UDP-based protocol with error recovery designed for low-latency streaming over unreliable networks.
Stinger
An animated video clip used as a scene transition in OBS, typically featuring motion graphics with transparency.
VBR
Variable Bitrate — encoder adjusts bitrate based on scene complexity. Efficient but can cause buffering on strict networks.
WebRTC
Web Real-Time Communication — browser-native protocol enabling sub-200ms streaming latency, used by Kick and video conferencing.
